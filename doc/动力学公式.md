## ⚠️ 运行前需要做的

### 1. 安装依赖

```bash
# 激活虚拟环境
source ijcai/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 准备数据（可选）

如果要测试真实数据集，需要：
- 下载 SNAP 数据集
- 将数据放在 `data/raw/` 目录下
- 支持格式：`.edgelist` 或 `.txt`

**注意**：训练时使用合成图，不需要真实数据。

### 3. 检查 GPU（可选）

如果有 GPU，可以：
- 安装 CUDA 版本的 PyTorch
- 在训练/测试时使用 `--device cuda`

## 🐛 常见问题

### 1. ImportError: No module named 'scipy'

**解决**：
```bash
pip install scipy
```

### 2. ImportError: No module named 'torch_geometric'

**解决**：
```bash
pip install torch-geometric
```

### 3. CUDA 相关错误

**解决**：
- 如果没有 GPU，使用 `--device cpu`
- 如果有 GPU，检查 CUDA 版本和 PyTorch 版本是否匹配

### 4. 数据文件未找到

**解决**：
- 训练时不需要真实数据（使用合成图）
- 测试时如果需要，下载 SNAP 数据集并放在 `data/raw/` 目录
