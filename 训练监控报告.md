## 🔍 监控功能

### 已创建的监控工具

1. **monitor_training.py** - 基础监控脚本
   - 检查模型文件
   - 读取训练日志
   - 显示最新状态

2. **auto_monitor.py** - 自动监控脚本
   - 持续监控训练进程
   - 检测训练是否卡住
   - 自动刷新状态

### 使用方法

```bash
# 单次检查
python scripts/auto_monitor.py --once

# 持续监控（每10秒刷新）
python scripts/auto_monitor.py --interval 10

# 监控指定次数
python scripts/auto_monitor.py --iterations 20
```
